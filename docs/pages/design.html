Design

Data Analysis Techniques

We employed exploratory data analysis techniques to identify trends visually. Furthermore, we used basic statistics to determine the min, max, modal and median values. This was easy thanks to the pandas built in functions. Frequency is not relevant in a dataset such as this as the average inflation rate of 2.75% would completely eliminate it.

Code Design

We used Python 3 to analyse the data. We chose snake_case as our variable and function naming convention throughout this project. We employed standard functional programming, we decided against an object oriented approach as it was not practical for an project of this scale. This project does not have custom datatypes or constructors that require this. This is because most of the heavy lifting is done by the panda library and matlib.pyplot library.

Types of graphs

We decided to go with line graphs as we thought it would display our data appropriately. The line graph allows us to easily identify spikes and dips in consumer spending. Furthermore, line graphs allow us to display 80 quarters in total spanning across 20 years on one graph. Something that can not be achieved on a bar graphs or pie chart for example.
